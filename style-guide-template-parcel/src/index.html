Style Guide 1.0

Purpose/Theme

Offer developing software programers an opportunity to see a polished website with various Css themes. It shows demonstrations of what can be accomplished through CSS-based design.The Zen Garden aims to excite, inspire, and encourage participation. The theme that we chose to do for our CSS Zen Garden project is that of Star Wars and Space in general. We plan on implementing this theme in a few different ways. We will use a background that looks like stars out in space, most likely created by svg. The majority of the body of the text will be in one larger section, that as the user scrolls through, it will disappear at the top and get smaller as if it is going into space (think introduction to the star wars movies). We would also lock the navigation links and other such content in place so that it gives more of the feel that the main body text is disappearing off the screen. We plan to implement more space ideas through the use of svg planets or other simple designs to add some other graphics. We may tilt some of our text and boxes in different, haphazard ways to help give the effect that the reader is looking at words that are floating in space. I believe that using a 3 columned grid approach will help us organize the content in a good way.
